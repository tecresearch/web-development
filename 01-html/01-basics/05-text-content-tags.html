Text Content Tags with Definitions

<p> (Paragraph Tag): The <p> tag is used to define a paragraph of text. It automatically adds space before and after the text.

This is a sample paragraph demonstrating the usage of the <p> tag.

<hr> (Horizontal Rule Tag): The <hr> tag is used to create a horizontal line, which is typically used to separate sections of content.

<pre> (Preformatted Text Tag): The <pre> tag is used to display text exactly as written in the HTML file, preserving whitespace and line breaks. 

        This text is displayed
        exactly as written in the HTML code.

<blockquote> (Blockquote Tag): The <blockquote> tag is used to define a block of quoted text, which is usually displayed with indentation.

"The best way to predict the future is to invent it." - Alan Kay

<ol> (Ordered List Tag): The <ol> tag is used to create an ordered (numbered) list. Each item is wrapped inside an <li> tag.

  1. First ordered item
  2. Second ordered item
  3. Third ordered item

<ul> (Unordered List Tag): The <ul> tag is used to create an unordered (bulleted) list. Each item is wrapped inside an <li> tag.

  • First unordered item
  • Second unordered item
  • Third unordered item

<dl> (Definition List Tag): The <dl> tag is used to create a list of terms and their definitions.

<dt> (Definition Term): The <dt> tag is used to define the term in a definition list.

<dd> (Definition Description): The <dd> tag is used to provide the description of the corresponding <dt> term.

HTML
HyperText Markup Language - the standard language for creating web pages.
CSS
Cascading Style Sheets - used to style and layout web pages.

<figure> (Figure Tag): The <figure> tag is used to contain images, diagrams, illustrations, etc.

<figcaption> (Figure Caption Tag): The <figcaption> tag is used to provide a caption for an image or other media inside a <figure>.

<div> (Division Tag): The <div> tag is a block-level container used to group elements together for styling or layout purposes.

This is a div element containing some text.

<span> (Span Tag): The <span> tag is an inline container used to apply styling or JavaScript actions to a specific section of text.

This is an example of a span tag inside a paragraph.
