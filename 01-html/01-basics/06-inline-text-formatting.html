Inline Text Formatting Tags with Definitions

<a> (Anchor Tag): The <a> tag is used to create hyperlinks, which allow users to navigate to different web pages or locations.

Visit Example Website.

<b> (Bold Text Tag): The <b> tag makes the enclosed text bold, but does not add semantic importance.

This is bold text.

<i> (Italic Text Tag): The <i> tag makes the enclosed text italic, usually for styling purposes.

This is italic text.

<u> (Underline Text Tag): The <u> tag underlines the enclosed text.

This is underlined text.

<strong> (Strong Tag): The <strong> tag makes text bold and conveys strong importance.

This is strong text.

<em> (Emphasized Tag): The <em> tag italicizes the text and provides emphasis, making it semantically important.

This is emphasized text.

<small> (Small Text Tag): The <small> tag reduces the size of the enclosed text.

This is small text.

<mark> (Marked Text Tag): The <mark> tag highlights text with a background color.

This is highlighted text.

<del> (Deleted Text Tag): The <del> tag represents deleted text with a strikethrough.

This is deleted text.

<ins> (Inserted Text Tag): The <ins> tag represents inserted text with an underline.

This is inserted text.

<sub> (Subscript Tag): The <sub> tag displays text slightly below the normal line.

H2O (Water formula)

<sup> (Superscript Tag): The <sup> tag displays text slightly above the normal line.

x2 + y2 = z2

<code> (Code Tag): The <code> tag displays programming code in a monospace font.

Example: console.log("Hello, World!");

<cite> (Citation Tag): The <cite> tag is used to reference the title of a book, movie, or work.
