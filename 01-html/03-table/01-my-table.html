Table Tags and Definitions

  • <table>: Defines a table.
  • <caption>: Specifies a title for the table.
  • <thead>: Groups header content inside a table.
  • <tbody>: Groups the main body content of a table.
  • <tfoot>: Defines the footer content of a table.
  • <tr> (Table Row): Represents a row inside a table.
  • <th> (Table Header): Defines a header cell.
  • <td> (Table Data): Defines a data cell.
